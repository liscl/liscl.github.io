Simple Google Maps API by using the exampe of a hiking tour in Burkhardswalde (next to Dresden)

The map shows a hiking route in Burkhardswalde. This route was mapped one year ago within a project. All the markers (except the Google Maps markers) were designed on my own and they provide some basic information about the places they mark. Working with Google Maps API is very easy and intuitive. It gives even non-cartographers who do not have any experiences with mapping the possibility to map their own routes. If you once figured out how to work with the Google Maps API, it is very easy to use it. It is easy as well to remember how to use it, even if you have not worked with it for a longer time.
