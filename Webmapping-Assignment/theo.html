Theoretical Project: Analysis of interactive hiking maps

suedtirol.com: This page offers an interactive web map of the area Suedtirol. It provides a lot of different funtions like different base maps (Outdooractive Summer, Satellite, Outdooractive Hybrid, Terrain, Outdooractive Winter, 3D-View), overlays (Panoramio, Slopes > 30°, 35°, 40°), Winter Sports, All routes, Long-distance hiking, cycling routes, long-distance cycling, MTB-Transalp) and different tours. You can search for a variety of tours in this area and query for different outdoor activities like hiking, biking, jogging, mountaineering, canyoning or horseback riding. The map also provides the functionality to display different points of interest and other information about e.g. traffic, shopping and events. In addition, it is possible to get information about accomodation or special hints about the hiking tours. The map offes some basic secondary functions like to switch between German, English an Italian or to display an information windows which explains how to use this map.


trekking.suedtirol.info: This web page offers an interactive map about the Suedtirol area as well. It provides a map view and areal images and a 2 or 3 dimensional map view as base maps. It is possible to plan, choose and search for a tour according to different parameters like region and category. Basic information about the routes are available as well. The map also gives information about e.g. the weather, how to hike with a GPS device or how to react in distress in the mountains. It is also possible to display a legend, switch between German, English and Italian, to display a elevation profile. If you display a tour, special information about length, needed time, differences in altitude and type of the path are provided next to the map in a small text box. The map has some special mouse over effects which show information and pictures about the point of interest. But mouse over effects are not provide for all points. In addition, it is possible to directly print the map from this web page.


zillertal.at: This web page offers interactive web maps about the region Zillertal. The landingpage links to different maps about the single areas. The maps have all the same layout and functionalities. The web map is a simple application and provides the different base maps: summer, winter, satallite and map. It is possible to zoom in and out and to move the map window. At the top of the page different signs are provided which enable switching on or off different types of tours (hiking, cycling, running). You can choose from different level of difficulties and an information window provides information about needed time, length of the tour, differences in altitude, an elevation profile and a describtion of the tour. In addition, the application gives the possibility to export a gpx-file of the tour. It is also possible to show different points of interest like accomondation, restaurants, sights or taffic. In addition, the map gives some panorama-pictures and offers links to live-cams.


vorarlberg.travel: This web page shows a map about the area Vorarlberg. It povides the nearly same base maps and overlays as the first map of Suedtirol: Outdooractive Summer, Outdooractive Hybrid, OpenStreetMap, Terrain, 3D-View, Panoramio, MTB-Transalp and different points of interest. As the map follows the same design and content guidelines as the first Suedtirol map, it offers nearly the same functionalities: searching for a variety of tours in this area and querying for different outdoor activities like hiking, biking, jogging, mountaineering, canyoning or horseback riding. If you select a tour, information about the length, needed time, differences in altitude and, a description of the tour, difficulty and many more are display. It is possible as well to get a 3D-flight and to export the tour as a gps-track.


Comparison of the web maps:
There are a lot of different interactive web maps available on the internet. Most of them share a variety of common functionalities. The four analyzed web maps all provide different base maps, including 3 dimensional map views. All the maps allow displaying and searching for different routes. Most of the maps provide additional information about the route, including an elevation profile and pictures. In addition, the maps show important points of interest. Some of the maps also allow to export the routes as gps-tracks. Not all maps provide the possibility to search for routes according to special parameters and not all maps distinguish between hiking, cycling and running routes. The maps are also differents according to the variety of offered functionalities. The most functions provides map 1 (Suetirol). Map 4 (Vorarlberg) has a lot of functions as well. The least functionalities offers map 2 (Suetirol Trekking) although it includes all the necessary functions. Most of the maps are adequate for the most users. They prodide the most inportant information and offer a variety of basic and easy understandible functions. All the functions are self-explaining and serve their purpose. The maps are all well designed and good to read. The symbology is satisfying and easy to understand. In conclusion, all the maps are user-friendly and absolutely easy and functional to use.
