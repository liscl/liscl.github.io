Webmapping Assignment 2

This web page was created for a project within the course "127.045 Internet-Kartographie" at the Vienna University of Technology (summer term 2013). The aim of this course accompanying website is to create a simple webpage which covers all the contents we learned in this course and to show that we are able to work with them.

The lecture covered the following sections: HTML, CSS, RSS, JavaScript, Google Maps API, OpenStreetMap, basic information about Geodata (standarts, file formats, data sources), usability and user interface design.

This web page provides subpages on the following topics:

  • Comparison of proprietary (Google Maps, Bing Maps, Yahoo Maps) and open source (OpenStreetMap) web mapping services
  • Simple Google Maps API by using the exampe of a hiking tour in Burkhardswalde (next to Dresden)
  • OpenStreetMap by using the example of "Schloss Weesenstein"
  • Usability: Gluecksatlas, Mapping Worlds
  • Theoretical project
