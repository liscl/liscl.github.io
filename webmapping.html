Webmapping-Assignment

This web page was created for a project during the second Semester of my Master studies. The aim of this course accompanying website is to create a simple webpage which covers all the contents we learned in this course. The lecture covered the following sections: HTML, CSS, RSS, JavaScript, Google Maps API, OpenStreetMap, basic information about Geodata (standarts, file formats, data sources), usability and user interface design.
